## 为什么选择 Astro？

Astro 的「群岛架构」让你可以在需要的地方使用交互组件，其余部分保持纯静态 HTML。这意味着更小的 JavaScript 包和更快的首屏加载。

## 核心优势

- **零 JS 默认**：页面默认不发送 JavaScript
- **内容集合**：类型安全的内容管理
- **框架无关**：支持 React、Vue、Svelte 等
- **内置优化**：图片优化、站点地图、RSS

## 实践建议

1. 使用 Content Collections 管理博客和项目
2. 通过 BaseLayout 统一 SEO meta 标签
3. 配合 Tailwind CSS 快速构建响应式 UI
4. 部署到 Vercel 或 Netlify，享受边缘 CDN 加速

## 总结

对于个人网站这类内容驱动的项目，Astro 是目前最平衡的选择——开发体验好、性能出色、部署简单。
