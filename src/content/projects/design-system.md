## 背景

随着产品线扩展，各业务团队的 UI 风格逐渐分化。我主导了设计系统的从 0 到 1 建设，统一视觉语言与交互规范。

## 成果

- 40+ 可复用 React 组件
- 完整的设计 token 体系
- Storybook 文档站点
- Figma 与代码双向同步流程

## 技术方案

采用 Monorepo 管理组件包，使用 CSS Variables 实现主题切换，通过 Changesets 管理版本发布。

## 影响

组件复用率提升 60%，新功能 UI 开发周期缩短约 40%。
