## 项目简介

一个零依赖的 Markdown 编辑器组件，可嵌入任何 Web 项目。支持 CommonMark 语法与 GitHub Flavored Markdown 扩展。

## 核心功能

- 实时分屏预览
- Shiki 语法高亮
- 快捷键支持
- 可定制主题
- 小于 15KB gzip

## 开源社区

项目在 GitHub 获得 500+ stars，被多个开源项目采用。欢迎提交 Issue 和 PR。

## 技术细节

使用 ProseMirror 作为编辑内核，通过虚拟 DOM diff 优化预览性能。
