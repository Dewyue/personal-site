## 项目背景

团队组件库缺乏统一文档，新成员上手成本高。本项目旨在建立「活文档」，让设计与开发共享同一套组件标准。

## 架构设计

- **Storybook**：交互式组件演示与 Props 文档
- **Astro**：静态指南页、设计 token 说明
- **Monorepo**：组件源码与文档同仓管理

## 成果

- 覆盖 40+ 基础与复合组件
- 设计 token 自动从 Figma 同步
- 文档站点 Lighthouse 性能分 98+

## 经验总结

文档站的价值在于「可被找到」。我们为每个组件配置了搜索关键词和使用场景，显著降低了内部支持成本。
